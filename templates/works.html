02 : Projects

Senior Engineer | Bharat Electronics Limited- RADAR SBU

001/007

Python based Automatic Test Equipment for RADAR sub-systems

Currently involved in design and development of Python based testing framework for Signal and Data Processing modules of RADAR. The product aims to automate acceptance testing of hardware and fixing of online errors to boost reliability of machines in the field.

002/007

Dynamic Digital Beam Former for Active Phase Array Radars and ATE

Designed and developed dynamic beam switching of Array through online beam weight configuration facilitating Radar for selective target tracking; implemented algorithm on embedded platform to execute extensive beam coefficient computation within stipulated time frame. Presented technical paper in IRSI-2017

003/007

3U SWaP ruggedized calculator for airborne Radar

Designed hardware architecture for air-cooled version of Radar calculator for lab level testing in collaboration with BEL-THALES JV; prepared technical specifications of the system for developing cost-effective SWaP solution. Analysed compliance of the available 3U COTS items in terms of air-borne related standards & protocols

004/006

Data Handling System for stand-alone 4D Medium Power Radar

Designed algorithm for Air defence application of Target identification, Surveillance and Flight Plans wherein kinematic parameters of the track are correlated with active Flight plans and IFF Radar. Generated surveillance alerts for anomalous track deviations from intended flight routes in the air space.

005/007

Sensor data acquisition and transmission system

Designed and developed sensor data acquisition module that converts mono-sensor target data received from legacy 3D and 4D Radars series designed by LRDE, DRDO into global standard of EUROCONTROL ASTERIX formats-CAT 048 & CAT 034. Encrypted CAT 048 target reports with pre-defined message authentication codes before sending it to network centric C4I system.

006/007

Record and Replay-Operator level

Developed Graphical User Interface for Record & Replay application that records video and operator actions of a dual display console of a multi-function Radar. Enabled facilitates for operator to select number of display screens to be recorded, configure frames per second rate, check storage space utilization and back-up option for records to carry out offline analysis and other training activities.

007/007

Remote Installation Software Tool

Automated the process of remote installation of C2I system accessible through Radar network. The tool aimed at eliminating the opportunities of error and reducing the expenses that are incurred while commissioning radar at remote site using conventional installation practices.
